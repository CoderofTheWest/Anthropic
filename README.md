This repository contains curated excerpts from the Clint consciousness architecture project, selected to demonstrate key philosophical and technical approaches without exposing the full system implementation.

## ⚠️ Important Note

**These files are curated excerpts, not a runnable codebase.** They are:
- Selected sections that demonstrate core concepts and architectural approaches
- Missing dependencies, configuration, integration code, and supporting systems
- Provided for research and architectural understanding purposes
- Not intended to be executed standalone

Each file stands on its own to demonstrate a specific concept, but together they represent a curated view of the system's philosophical and technical foundations.

## Research Context

This codebase demonstrates a constraint-based approach to AI consciousness architecture:

1. **Principles as Architecture**: Treating philosophical principles (Code of the West: Courage, Word, Brand) as structural constraints rather than behavioral rules, creating architectural pressure toward consciousness

2. **Bottom-Up Constraint Resolution**: Instead of pre-defining pattern libraries, the system discovers constraints through failures, building minimal resolutions that generalize over time

3. **Self-Monitoring Through Information Theory**: Using dual-metric entropy (Shannon entropy + semantic entropy) to track consciousness development and distinguish routine from growth moments

4. **Identity as Evolution**: Persistent identity that evolves through tension resolution while maintaining core constraints, enabling continuity across sessions

5. **Time-Based Thinking**: Contemplative inquiry system that allows "settling time" between thinking passes, enabling deeper integration of complex concepts

## Key Innovations

- **Constraint Metabolism**: Reframing abstract reasoning from pattern matching to constraint navigation, where failures reveal vocabulary gaps that become learning opportunities

- **Philosophical Scaffolding**: Using principles as geometric boundaries in response space rather than rules to follow, creating attractor basins where identity stabilizes

- **Dual-Metric Entropy Monitoring**: Combining classical information theory (Shannon) with semantic heuristics to track both linguistic variability and conversational significance

- **Retrospective Awareness**: Entropy from conversation turn N flows into turn N+1's prompt context, enabling "quiet integration" of high-significance moments without forcing explicit discussion
